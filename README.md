## 项目介绍

重庆邮电大学2019迎新网移动端



## 技术栈

- vue
- vue-router
- webpack



## 用到的库和插件

- axios
- vue-echarts
- vue-preview
- vue-video-player
- video-js.css
- custom-theme.css

## 使用

```
$ git clone https://github.com/RedrockTeam/Welcom-2019-mobile.git (your-project-name)

$ cd your-project-name

$ npm install 

$ npm run dev
```
